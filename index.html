Implementation

This is the implementation of the “Face Detection, Pose Estimation, and Landmark Localization in the Wild” algorithm by Xiangxin Zhu and Deva Ramanan. This implementation was firstly designed for being used by embedded systems but finally it can also be used by large multiprocessors systems. Our implementation is in basic C\C++ and there is no usage of any external library in the core of the algorithm. This gives the algorithm the ability to be used in both Windows and UNIX systems with no further changes. It also allows further improvements and alteration as it is easily readable for those who would like to use it for custom application.

Documentation

This implementation was created as a master thesis in the Technical University of Crete by Panos Kalodimas and its documentation can be found here. The implementation offers two extra patches as referred in the documentation chapter 6 (Default Patches). The rest patches presented in chapter 8 (Alternative Patches) are not included. In the documentation the memory consumption and the execution time performance can also be found for embedded system usage.
